# Что даёт 1REG Prime

Prime — платная подписка в 1REG, которая расширяет возможности бота.

## Что входит в Prime

| Функция | Без Prime | С Prime |
|---|---|---|
| Регистрация доменов | ✅ | ✅ |
| Смена NS и DNS | ✅ | ✅ |
| Cloudflare в 1 клик | ✅ | ✅ |
| Автопродление доменов | ❌ | ✅ |
| Авто-проверки блокировок | ❌ | ✅ |
| Уведомления о блокировках | ❌ | ✅ |
| Расширенный API-доступ | ❌ | ✅ |

## Для кого Prime особенно полезен

- Если у вас много доменов и важно не пропустить продление
- Если нужны уведомления о блокировках в реальном времени
- Если работаете через API и нужны расширенные возможности
