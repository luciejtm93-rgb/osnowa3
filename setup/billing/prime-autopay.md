# Как работает Prime и автопродление

## Как оформить Prime

1. Откройте бота [@one\_reg\_bot](https://t.me/one_reg_bot)
2. Перейдите в раздел **Prime**
3. Выберите период подписки
4. Подтвердите оплату с баланса

Prime активируется сразу после оплаты.

## Как работает автопродление

После активации Prime бот автоматически продлевает домены до истечения срока — деньги списываются с баланса.

**Важно:** на балансе должно быть достаточно средств. Если баланс пустой — автопродление не сработает и домен может истечь.

→ [Как пополнить баланс](top-up.md)

## Как проверить что автопродление включено

Откройте бота → **Мои домены** → выберите домен → убедитесь что автопродление активно в карточке домена.

## Prime + API

С Prime открывается расширенный доступ к API — можно регистрировать домены и управлять ими программно.

{% hint style="warning" %}
Prime — подписка на аккаунт, а не на отдельный домен. При переносе аккаунта уточните у поддержки статус подписки.
{% endhint %}
