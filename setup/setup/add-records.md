# Как добавить A, CNAME, TXT или MX

## Пошагово

1. Откройте бота [@one\_reg\_bot](https://t.me/one_reg_bot)
2. Перейдите в **Мои домены** и выберите домен
3. Нажмите **DNS**
4. Нажмите **Добавить запись**
5. Выберите тип записи: A, CNAME, TXT, MX или другой
6. Заполните поля и сохраните

## Какую запись добавить

**A-запись** — направляет домен на IP-адрес сервера.
Пример: `@` → `185.10.20.30`

**CNAME** — направляет домен или поддомен на другой домен.
Пример: `www` → `mysite.example.com`

**TXT** — текстовая запись для верификации сервисов (Google, почта, Brevo).
Пример: `@` → `v=spf1 include:... ~all`

**MX** — указывает сервер входящей почты.
Пример: `@` → `mail.example.com`, приоритет `10`

## Запись для www

Чтобы домен открывался и без www, и с www — добавьте две отдельные записи:

- A-запись: `@` → ваш IP
- A-запись: `www` → ваш IP

## Сколько ждать

DNS-записи обновляются обычно в течение 5–30 минут.
