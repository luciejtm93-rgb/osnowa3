# Что такое 1REG DNS и когда лучше Cloudflare

В 1REG есть два варианта управления DNS — собственные NS и Cloudflare. Вот чем они отличаются.

## Сравнение

| | 1REG DNS | Cloudflare |
|---|---|---|
| Простота настройки | ✅ | ✅ (1 клик) |
| SSL автоматически | ❌ | ✅ |
| Работает в России | ✅ | ⚠️ нестабильно |
| CDN и защита от DDoS | ❌ | ✅ |
| IP сервера скрыт | ❌ | ✅ |

---

## 1REG DNS (базовые NS)

Простое управление DNS прямо в боте. Вы добавляете A, CNAME, TXT, MX-записи — и домен работает.

**Подходит если:**
- Нужно просто направить домен на сервер
- Используете хостинг, который управляет записями сам
- Работаете в России и важна стабильная доступность

---

## Cloudflare в 1 клик

1REG подключает Cloudflare автоматически — NS домена меняются на Cloudflare, SSL активируется сам.

**Подходит если:**
- Важна безопасность, скорость и SSL без ручной настройки
- Нужно скрыть IP сервера

**Учтите:**
- В России Cloudflare может работать нестабильно или не работать совсем
- После подключения управление DNS переходит в Cloudflare

---

## Как подключить Cloudflare

1. Откройте бота [@one\_reg\_bot](https://t.me/one_reg_bot)
2. Перейдите в **Мои домены** → выберите домен
3. Нажмите **Cloudflare в 1 клик**
4. Подтвердите подключение

SSL активируется автоматически в течение нескольких минут после подключения.
