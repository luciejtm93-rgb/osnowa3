# Почему у меня не открывается, а у других открывается

Домен работает у коллеги или на телефоне, но не открывается у вас — проблема в локальном DNS-кеше или провайдере.

## Причина

Ваш компьютер или провайдер хранит старые DNS-данные дольше других. Это не баг — просто ваш кеш ещё не обновился.

## Что сделать

**1. Откройте в режиме инкогнито**
Иногда достаточно этого.

**2. Попробуйте с другого устройства или через мобильный интернет**
Если там открывается — проблема точно в локальном кеше.

**3. Сбросьте DNS-кеш**

Windows:
```
ipconfig /flushdns
```

Mac:
```
sudo dscacheutil -flushcache && sudo killall -HUP mDNSResponder
```

**4. Смените DNS-сервер**
Попробуйте публичный DNS — Google `8.8.8.8` или Cloudflare `1.1.1.1`. Инструкция есть в настройках сети вашей системы.

## Если домен не открывается только в России

Возможно, включён Cloudflare proxy — он может работать нестабильно для российских пользователей. Попробуйте отключить Cloudflare proxy в боте.
