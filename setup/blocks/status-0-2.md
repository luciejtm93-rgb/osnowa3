# Что значит 0/2

В карточке домена в боте вы можете увидеть статус вида `0/2` или `1/2` в разделе блокировок.

## Что означают цифры

Формат: **активные блокировки / всего проверок**

| Статус | Что значит |
|---|---|
| `0/2` | Проверки прошли, активных блокировок нет |
| `1/2` | Одна из проверок выявила блокировку |
| `2/2` | Обе проверки выявили блокировки |

## Если статус 0/2

Всё хорошо — домен не заблокирован по результатам автоматических проверок.

Если при этом домен не открывается — причина не в блокировке.
→ [Домен не открывается: быстрая проверка](../not-working/quick-check.md)

## Если статус 1/2 или 2/2

Домен попал под одну или обе проверки. Откройте карточку домена в боте — там будет указана причина.

→ [Client Hold / Server Hold](client-server-hold.md)
