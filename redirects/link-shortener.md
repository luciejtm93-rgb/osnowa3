# Как работает сокращение ссылки

С помощью сокращения ссылок вы можете использовать свой домен как короткий URL — например, `yourdomain.com/go` вместо длинной ссылки.

## Пошагово

1. Откройте бота [@one\_reg\_bot](https://t.me/one_reg_bot)
2. Перейдите в **Мои домены** и выберите домен
3. Нажмите **Сокращение ссылок**
4. Введите короткий путь и URL назначения
5. Сохраните

## Как это выглядит

Вы создаёте правило: `yourdomain.com/promo` → `https://long-url.com/campaign/123`

Когда кто-то открывает `yourdomain.com/promo` — его автоматически перенаправляет на нужный URL.

## Что важно учесть

- Функция работает только если NS домена направлены на 1REG DNS
- На одном домене можно создать несколько коротких ссылок
- При передаче домена другому аккаунту ссылки переходят вместе с ним
