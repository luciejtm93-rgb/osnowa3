# Что такое совладельцы

Совладелец — это другой пользователь 1REG, которому вы дали доступ к управлению вашим доменом.

## Чем совладелец отличается от передачи домена

| | Совладелец | Передача домена |
|---|---|---|
| Кто остаётся владельцем | Вы | Получатель |
| Кто управляет доменом | Оба | Только получатель |
| Можно отозвать | ✅ | ❌ |

## Как добавить совладельца

1. Откройте бота [@one\_reg\_bot](https://t.me/one_reg_bot)
2. Перейдите в **Мои домены** → выберите домен
3. Найдите раздел **Совладельцы**
4. Добавьте Telegram ID или username пользователя

## Как убрать совладельца

Зайдите в карточку домена → раздел **Совладельцы** → удалите пользователя из списка.

## Сколько совладельцев можно добавить

Количество совладельцев указано в карточке домена в поле **Совладельцев: N**. Точный лимит уточните у поддержки.

{% hint style="warning" %}
Совладелец имеет доступ к управлению NS, DNS и настройками домена. Добавляйте только тех, кому доверяете.
{% endhint %}
