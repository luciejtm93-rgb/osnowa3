# Почта, язык и профиль

## Как привязать почту

Привязанная почта нужна для восстановления доступа к аккаунту если вы потеряете Telegram.

1. Откройте бота [@one\_reg\_bot](https://t.me/one_reg_bot)
2. Перейдите в **Профиль** или **Настройки**
3. Нажмите **Привязать почту**
4. Введите email и подтвердите

{% hint style="info" %}
Привяжите почту заранее — это единственный способ восстановить доступ к доменам если Telegram-аккаунт будет заблокирован.
{% endhint %}

## Как поменять язык

1. Откройте бота [@one\_reg\_bot](https://t.me/one_reg_bot)
2. Перейдите в **Настройки**
3. Выберите **Язык**
4. Выберите нужный язык из списка

---

## Как посмотреть профиль

В профиле отображаются:
- Ваш ID в системе
- Баланс
- Статус Prime
- Привязанная почта
- Данные WHOIS (если заполнены)

Чтобы открыть профиль: бот → **Профиль** или команда `/start` → раздел профиля.
